## Install and use

<br />

✅ yarn 설치 방법 (독립형 스크립트)

---

<br />


✅ 의존성 설치 (<strong>pnpm `설치 작업` 선행 후 진행</strong>)
> npm이 아닌 pnpm을 사용합니다. ( https://pnpm.io/ko/installation )

```bash
yarn install
```

<br />

✅ 프로젝트 기동 (의존성 설치 후 진행 가능)
```bash
yarn dev
```

<br />

✅ 프로젝트 빌드
```bash
yarn build
```

<br />

✅ 의존성 재설치
```bash
yarn reinstall
```


<br />

✅ dependencies에 추가
```bash
yarn add (명칭)
```

<br />

✅ dev devDependencies에 추가
```bash
yarn add -D (명칭)
```
